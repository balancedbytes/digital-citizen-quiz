# What Kind of Digital Citizen Are You? - Quiz

This repository contains a lightweight, stylish HTML quiz designed by the blog **Secured Wellness** to help users reflect on their digital habits across personal digital security and digital wellness.

---

## About the Quiz

- **12 quick questions** designed to assess your digital security and wellness habits.
- Provides a personalized result with helpful tips and next steps.
- Mobile-friendly and minimalist design with pastel colors.
- Includes a “Share Your Result” button and a CTA button linking to [securedwellness.net](https://securedwellness.net).

---

## How to Use

1. The quiz is a single HTML file (`index.html`).
2. Open `index.html` directly in any modern web browser to take the quiz offline.
3. To host the quiz online, use GitHub Pages or any static file hosting service.

---

## Hosting with GitHub Pages

1. Create a GitHub repository and upload `index.html`.
2. In the repository settings, enable GitHub Pages from the `main` branch root directory.
3. Access the live quiz at `https://yourusername.github.io/your-repo-name/`.

---
